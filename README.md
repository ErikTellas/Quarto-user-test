# Quarto mall

See siin on Quarto raamatu mall. Selle abil saab luua raamatu, mis sisaldab juba GitHub Actions töövoogu, mis loob raamatust ka automaatselt veebilehe. Raamatu saab luua, klõpsates nupule **Use this template** ja seejärel **Create a new repository**.

Lisaks on paigaldatud Google Analytics, mis võimaldab jälgida, kui palju inimesi su raamatut külastab ja kuidas nad sellega suhtlevad. Vaata faili `analytics.html` ja asenda `G-XXXXXXXXXX` oma Google Analytics Measurement ID-ga, et see tööle hakkaks.
